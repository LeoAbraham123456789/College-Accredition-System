# It was created using Eclipse, so it would be better if you use Eclipse.

# Steps to execute:
    1) Open the Mini-Project folder in Eclipse
    2) Run the java Project

# 1st window is the login window, from there you can go to 
    a) departments(now for the time being only cs, ec, mech are only given)
    b) page for the principal

# In the departments page ther is option for ticking those steps that 
    the respective departments have finished. And also a button for updating and refreshing the page.
    Note: Only after refreshing, you must update the details.

# In the principal page you can see the progress of each department
    there is a button for validation,
    when it is clicked a file for that department whose validate button is pressed 
    is written in the background.
